FAQ’S

Q1. How can I uninstall the Canon IJ Scan Utility software from your computing device?

Ans. You can uninstall the Canon IJ Scan utility software, in case it is not running properly or you have to install the upgraded version of it.

  • The preliminary step is to tap on the Start on windows icon as per the option that is currently available on your computer or laptop. 
  • Soon after this, locate and press the Control Panel option from the right-hand panel. This will open the Control Panel window right in front of you. 
  • Thereafter, select the Programs option and after this go to Programs And Features. You will find all the software that is being installed in your system. 
  • To install the IJ Scan Utility software, you have to right-click on the printer driver that you have previously installed. After that, press the Uninstall option from there. 
  • Lastly, it removes the software from your system, and then you can install the newer or upgraded version of it to scan your documents easily. 

Q2. How can you set the documents scan settings into the IJ Scan Utility Software?

Ans. In order to set the documents scan settings option within this software, you need to examine the following steps closely.

  • In the beginning, go to the Documents Scan Settings page within the Canon IJ Scan Utility software. From there you need to opt for the Settings option. 
  • As a result, Settings Window appears, tap on the Documents Scan option from the left-hand panel. 
  • By doing so, it will instantly open the Settings(Documents Scan) dialogue box on the screen. Now, from the following window, you can easily apply the upgraded scan settings option. 
  • In the following dialogue box, you will get notice three different sections that you have to set for scanning the documents. These three sections are Scan Options Area, Save Settings Area, and Applications Settings Area. 
  • Next, under the Scan Options Area, you can fix the opt for the source that you want to scan, opt for the color mode, set paper size, image processing settings, etc. You can manage all of these settings under this segment. 
  • Thereafter, under the Save Settings Area segment, you have to name the file that you want to scan, choose the data format, Check scan results, and save in option. 
  • Finally, beneath the segment of Applications Settings Area, you will get different options for correct the scanned images, attach an email, and other functions like these. 
  • At last, once you have added all these settings, you can move further to scan the documents or images using IJ Scan Utility software.
